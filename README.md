# Image Gallery Project

## Description
This project is a responsive image gallery built using HTML, CSS, and JavaScript.

## Features
- Image grid layout
- Hover zoom effect
- Lightbox (click to view image)
- Next and Previous navigation
- Category filtering (Food, Nature, Car)
- Responsive design

## Technologies Used
- HTML
- CSS
- JavaScript

## How to Run
1. Open index.html in a browser
2. Click images to view
3. Use filter buttons to sort images
